# YADRO Contacts App

Android-приложение для просмотра контактов, сохранённых на устройстве, и исходящего звонка по нажатию на контакт в списке.

## User Stories

### US1. Список контактов устройства

**Как пользователь**, я хочу видеть список контактов своего устройства.

**Описание:** приложение отображает контакты с телефона в виде списка. Каждый пункт содержит краткую информацию о контакте.

**Критерии приёмки:**

| Критерий | Реализация |
|----------|------------|
| В списке только контакты с устройства | `ContentResolver` + `ContactsContract.CommonDataKinds.Phone` в `DeviceContactsDataSource` |
| Краткая информация о каждом контакте | Имя, основной номер телефона, аватар |

Дополнительно:

- список сгруппирован по первой букве имени (контакты без имени — в секции `#`);
- в списке один пункт на контакт, номер — **main (primary)** из адресной книги (`IS_PRIMARY = 1`);
- контакты без primary-номера не попадают в список;
- при отсутствии разрешения `READ_CONTACTS` список пустой.

### US2. Звонок по нажатию на контакт

**Как пользователь**, я хочу позвонить любому контакту из списка по нажатию на пункт.

**Описание:** по тапу на элемент списка выполняется звонок на номер, который отображается в этом пункте (main phone number).

**Критерии приёмки:**

| Критерий | Реализация |
|----------|------------|
| Звонок по клику на пункт списка | `clickable` на элементе списка → `MainScreenViewModel.onCallRequested()` |
| Звонок на номер из пункта списка | `Intent.ACTION_CALL` с `tel:`-URI того же номера, что показан в UI |

При отсутствии `CALL_PHONE` запрашивается разрешение; после выдачи выполняется отложенный звонок.

Запрос при первом запуске; при отказе — диалог с rationale и переход в настройки приложения.
